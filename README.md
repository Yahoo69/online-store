 # 🛒 Simple Online Store (PHP)

This is a online store built using PHP and MySQL. It allows users to browse products, add them to a cart, and place orders.

## 💡 Features

- Product listing
- Shopping cart
- Order form

## ⚙️ Requirements

- PHP 7.4 or higher
- MySQL
- Apache or Nginx server (XAMPP, MAMP, etc.)
